# My Blog Website

This is a personal blog website built using Django and Bootstrap.

## Features
- User-friendly interface
- Responsive design with Bootstrap 4.4.1
- Blog post creation, editing, and deletion
- User authentication (Login/Registration)
- Commenting system
- Search functionality

## Technologies Used
- Django
- Bootstrap v4.4.1 ([Official Website](https://getbootstrap.com/))
- HTML, CSS, JavaScript
- SQLite / PostgreSQL (Database)

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/zarnigor/blog-site.git
   ```
2. Navigate to the project directory:
   ```sh
   cd blog-site
   ```
3. Install dependencies using Poetry:
   ```sh
   poetry install
   ```
4. Apply database migrations:
   ```sh
   python manage.py migrate
   ```
5. Run the development server:
   ```sh
   python manage.py runserver
   ```
